# `components/brand`

Canonical Planton brand marks, drawn inline with `currentColor` so they inherit
text color and stay crisp at any size.

- `PlantonMark` — the "P" glyph (shared with planton.ai). The brand mark of record.
- `Wordmark` — the "Planton" text lockup in the display face.
- `DiscordIcon` — the official Discord glyph (reused from planton.ai).

Rule: never invent or substitute a brand mark. If a new brand asset is needed
and not available, stop and request the real file.
